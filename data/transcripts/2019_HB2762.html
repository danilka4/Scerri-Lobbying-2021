Firefighting foam management; use of foam that contains PFAS chemicals. (HB2762)

Introduced By

Del. David Bulova (D-Fairfax) with support from co-patron Del. Tony Wilt (R-Harrisonburg)

Description

Firefighting foam management. Prohibits, beginning July 1, 2021, the use for training purposes or for testing, with some exceptions, of a class B firefighting foam that contains intentionally added PFAS chemicals, as defined in the bill. Read the Bill »
